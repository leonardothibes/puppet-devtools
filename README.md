puppet-devtools
===============

A Puppet module that installs all necessary packages for compiling basic software.

## Sample Usage
Install devtools and use the provided configuration defaults
```
node default {
	class {'devtools':}
}
```
or
```
node default {
	include devtools
}
```

Uninstall devtools
```
node default {
	class {'devtools':
		ensure => absent,
	}
}
```
